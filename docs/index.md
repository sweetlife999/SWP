# SWP — Student Union Portal Documentation

Welcome to the maintained documentation for the **Student Union Portal (SWP)**. This
site publishes the project's living documentation — architecture, development process,
quality, and testing — as browsable pages so reviewers can inspect the current state of
the product without reading raw repository paths.

The pages here are generated from the Markdown sources under [`docs/`](https://github.com/sweetlife999/SWP/tree/main/docs)
in the repository, and every document is reachable from the navigation sidebar.
